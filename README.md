# Horiseon
Horiseon empresa de marketing digital con sede la ciudad de Guadalajara, Jalisco México.

Este proyecto busca proporcionar una plantilla gratuita simple y funcional para el inicio de su proyecto de página web.


## Estilo de codificación
Buscamos mantener una página web que cumpla con estándares de accesibilidad cumpliendo lo siguiente:

- Cuidando los elementos semánticos de HTML.
- Manteniendo secuencia lógica e independiente de estilo y posicionamiento.
- Utilizando y manteniendo selectores y propiedades CSS consolidados y organizados.
